# Alura_Cookin-Up_Vuejs
Curso da Alura aprofundando em Vue.js

 PASSOS PARA INSTALAR VUE.JS

npm create vue@numero da versão desejada ou só
npm create vue para a versão mais recente

depois

   cd cookin-up
   npm install
   npm run dev
